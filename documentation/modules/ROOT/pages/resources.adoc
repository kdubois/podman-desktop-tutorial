= Workshop Summary and Resources
include::_attributes.adoc[]

Congratulations on completing the Podman Desktop workshop! In this workshop, you learned how to:

- Build and manage container images using Podman Desktop
- Run and manage containers using Podman Desktop
- Create pods from containers using the "Podify" feature
- Deploy pods to Kubernetes using Podman Desktop
- Scale and manage deployments in Kubernetes

You now have a solid foundation in using Podman Desktop for containerized application development and deployment, and we encourage you to attend the more technical Podman Desktop sessions during Summit to learn more!
